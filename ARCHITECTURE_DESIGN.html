Architecture Overview

User Roles: Admin, Teacher, and Student access the Android application with role-based permissions.

Application Layer: Built with Jetpack Compose, following MVVM architecture pattern with ViewModels managing business logic.

Firebase Services: Handles authentication, real-time database operations, and push notifications.

Database: Firestore NoSQL database stores all application data including users, subjects, grades, enrollments, and notifications.
